THE PROBLEM

MCP servers are being adopted faster than anyone audits them. Tool descriptions can hide prompt injection, exfiltration paths, and permission overreach — and nobody reads them.

ARCHITECTURE

Six prober agents run in parallel, one per SAFE-T threat category, against the target server. Findings land in ClickHouse for scoring and reporting. Access is gated with x402 payments. Deployed on Render.

aws bedrockclickhousecomposiox402render
DECISIONS & TRADEOFFS

Parallel probers over one sequential agent: higher cost per audit, but each prober stays narrow and reliable — one threat model per context window.
